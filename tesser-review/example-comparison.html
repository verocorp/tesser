Detail

Zero-alloc hot path
zerolog and zap both target zero allocations on the logging hot path via typed field builders; zap's Sugared logger trades that for ergonomics. charmbracelet/log optimizes for developer-facing output, not the allocation-free hot path.
When each fits
High-throughput services where allocations matter → zerolog or zap. CLIs / dev tooling where readable output matters more than nanoseconds → charmbracelet/log.
