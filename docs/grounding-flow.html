  P1 -->|resolved + pinned| P2
  P1 -->|"404 · recalled → re-search"| P1
  P1 -->|"404 · dev-given (after S8)"| TR["can't reach"]:::warn
  P1 -. "search empty / no pick" .-> ASKURL{{"ASKURL — repo URL?"}}:::event
  ASKURL -->|URL| P1
  ASKURL -->|doesn't know| TU

  P2["P2 · Reuse — reuse-first, offline

  P2 -->|"map-hit (SKIP P3)"| P4
  P2 -->|clone-hit / miss| P3

  P3["P3 · Gather — climb docs→inspect→run

  P3 -->|each rung feeds the join| P4
  P3 -. "corroboration mismatch" .-> CMATCH{{"CMATCH — right repo?"}}:::event
  CMATCH -->|yes| P3
  CMATCH -->|"no (back-edge)"| P1
  CMATCH -->|gives URL| P1
  P3 -->|build/run nonzero| TCU["completed-unverified"]:::warn

  P4["P4 · Join — the synthesizer

  P4 -->|"higher grade changes answer (beat-2, async)"| P4
  P4 -->|joined + gathering done| TA["answered · docs / inspect / run"]:::good
